## 使用方法

範例：
> 使用者輸入：@哈哈狗 那你很厲害誒\
> 哈哈狗回覆：那你很厲害誒😎😎😎

1. 在 LINE 上加哈哈狗好友：[@255eanep](https://lin.ee/teUKO7u)\
    ![QR Code](./qr-code.png)
2. 向哈哈狗傳送帶有前綴或後綴 `@哈哈狗` 的文字。\
   例如： "@哈哈狗 我愛你" 或 "我愛你 @哈哈狗"
3. 哈哈狗也可以加入群組使用！
